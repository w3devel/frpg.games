Features

🎨

Scriptable Network Graphics (SNG)

Create 2D tiles and isometric 3D cubes programmatically using SNG — a scriptable graphics engine that produces PNG and APNG output without requiring a full rendering pipeline.

🧠

Prolog Game Logic

A built-in Prolog engine evaluates game rules in real time. Query attack modifiers, spell slot counts, saving throw DCs, or any game mechanic directly from the editor.

⬡

Hex & 3D Polyhedra

Go beyond square grids. Design maps using hexagonal tiles or 3D polyhedron honeycombs. Miniatures render correctly for any view: top-down, isometric, or perspective.

🎞

Animated Sprites (APNG)

Animate your assets frame by frame. Frames are assembled into Animated PNG (APNG) files using apngasm compiled to WebAssembly, running in a Cloudflare Worker for browser-side assembly.

☁

Cloudflare Worker Backend

Heavy processing — such as multi-frame APNG assembly — runs in a Cloudflare Worker. No server installation required. Assets stay private in your extension storage.

🔌

Browser Extension

The extension (Manifest V3, Chrome & Firefox) gives you persistent asset storage, Cloudflare Worker access, and seamless integration with the frpg.games editor even when offline.

Game Systems

FRPG.games is system-agnostic at its core. Game-specific rules are loaded as Prolog rule sets. Currently supported:

D&D SRD 5.2.1

CC BY 4.0

Dungeons & Dragons 5th Edition System Reference Document (SRD 5.2.1) published by Wizards of the Coast under Creative Commons Attribution 4.0.

  • Ability modifiers & proficiency bonuses
  • Attack rolls, saving throws, critical hits
  • Spell slot tables (levels 1–11)
  • Combat conditions, cover, exhaustion
  • 14 creature types with combat shapes

Pathfinder 2e

ORC License

Pathfinder Second Edition by Paizo Inc., published under the ORC (Open RPG Creative) License.

  • Proficiency ranks (Untrained → Legendary)
  • 4-degree success system (crit success/fail)
  • 3-action economy
  • Multiple Attack Penalty (MAP & Agile)
  • 10 ancestries with combat templates

Black Flag RPG

CC BY 4.0

Black Flag Roleplaying by Kobold Press, published under Creative Commons Attribution 4.0.

  • Fortune Die mechanic
  • Lineages & Heritages
  • Talents system
  • Familiar D&D-compatible mechanics
  • 8 lineages with speed & size data

Additional systems can be added by loading a Prolog rule file. The engine is designed to support any ORC, CC, or similarly licensed system.

Technology

SNG Pipeline

SNG Script (JSON)
  → SngBuffer (RGBA pixels)
    → OffscreenCanvas
      → PNG (ArrayBuffer)
        → APNG (via Worker)

Prolog Query Example

% Query proficiency bonus at level 5
?- proficiency_bonus(5, B).
B = 3.

% Fireball spell slot check
?- spell_slots(7, 3, S).
S = 3.

Cloudflare Worker

POST /apng
  form: frame0.png, frame1.png…
  form: delays=[100,100,…]
  form: loops=0
→ animation.apng

Browser Extension

The FRPG.games browser extension unlocks full functionality:

  • 💾 Persistent asset storage (chrome.storage)
  • ☁ APNG assembly via Cloudflare Worker
  • 🔗 Seamless sync between editor and frpg.games
  • ⌨ Keyboard shortcut: Ctrl+Shift+E to open editor

To install manually (developer mode):

  1. Clone / download this repository
  2. Open chrome://extensions
  3. Enable "Developer mode"
  4. Click "Load unpacked" and select the repository folder
